Pre-requisite:
  • HHA Enterprise Visit Log Export Abilities
  • Download Visit Log from HHA by ( Report -> Visits -> Visit -> Visit Log, Change Your Billing Parameters, Then View Report -> Top Right(Save Icon Besides Next Button) -> CSV Save To Computer)
  • Click The "Import Visit Log Button" -> Select The Downloaded File -> Open
Useful Search Hints:
  • You Can Search Via, Caregiver Name, Clients Name, Etc.. (Anything Thats On the Table)
  • You Can Use AND Symbol ("&") to Isolate Specific Cases, As in "Caregiver Name&Clients Name"
  • You Can Use OR Symbol ("|") to Isolate More Elements, As in "Caregiver Name|Clients Name"
  • Search "MIO" for Missed In and Out Visits
  • Search "Incomplete" To See if the Clock In Hours Are Not Equal To Scheduled Hours
  • Search "Un-billable" To See Just Missed In, Missed Out Visits
